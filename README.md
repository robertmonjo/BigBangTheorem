# BigBangTheorem

Lean supplementary material for the article on the Big Bang closedness theorem.

Files:

- `BigBangTheorem/BigBangTheorem_Strong.lean`: strong version based on Lipschitz control of the slice-volume function.
- `BigBangTheorem/BigBangTheorem_Weak.lean`: weak version based on integrable accumulated expansion.

Build:

```bash
lake build
```

The project uses Lean `v4.29.0-rc8` and a pinned version of `mathlib4`.
